# Student Productivity & Task Tracker

A simple Python-based console application designed to help students manage daily tasks efficiently.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using a JSON file

## Technologies Used
- Python
- File Handling (JSON)
- Basic Data Structures

## How to Run
1. Clone the repository:
git clone https://github.com/Gowshika-K/student-productivity-tracker.git

2. Navigate to the project folder:


cd student-productivity-tracker

3. Run the program:


python main.py
